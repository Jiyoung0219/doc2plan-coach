# 📄 Doc2Plan Coach
AI Agent 기반 과제·프로젝트 문서 분석 및 계획 생성 프로토타입

---

## 1. 프로젝트 개요

Doc2Plan Coach는 **과제 안내 PDF 또는 프로젝트 가이드 문서**를 업로드하면  
문서에 포함된 요구사항과 평가 기준을 자동으로 구조화하고  
이를 바탕으로 **개인 과제 코칭**과 **팀 프로젝트 PM 계획**을 생성해주는  
AI Agent 기반 서비스 프로토타입입니다.

대학 과제, 캡스톤 디자인, 팀 프로젝트 등에서  
“무엇을 해야 하는지 정리하기 어렵다”는 문제를 해결하는 것을 목표로 합니다.

---

## 2. 문제 정의

과제나 프로젝트 안내 문서는 대부분 PDF 형태로 제공되고

- 요구사항이 글로 흩어져 있음
- 평가 기준이 명확히 구조화되어 있지 않음
- 팀 프로젝트의 경우 역할 분담과 일정 계획이 어렵다는 문제가 있음

특히 입문자나 프로젝트 경험이 적은 학생의 경우  
**문서를 이해하는 데에만 많은 시간이 소요**됩니다.

---

## 3. 해결 방안

본 프로젝트는 Upstage의 Document AI 및 LLM API를 활용하여 다음과 같은 흐름으로 문제를 해결합니다.

1. **Document Parse**
   - PDF 문서를 AI가 읽을 수 있는 형태로 변환
2. **Information Extract (IE)**
   - 문서에서 과제 요구사항, 제출물, 평가 기준 등을 JSON 스키마로 구조화
3. **Solar 기반 AI Agent**
   - 구조화된 정보를 바탕으로
     - 개인 과제 체크리스트 및 수행 계획 생성
     - 팀 프로젝트 역할 분담, 일정, 리스크 분석 생성
     - 사용자가 작성한 초안에 대한 피드백 제공

이 과정을 통해 사용자는 문서를 직접 분석하지 않아도  
**바로 실행 가능한 계획과 가이드를 얻을 수 있습니다.**

---

## 4. 핵심 기능

### 4.1 개인 과제 코치
- 과제 안내 PDF에서 요구사항 자동 추출
- 평가 기준 기반 체크리스트 생성
- 사용자가 작성한 초안에 대한 피드백 제공

### 4.2 팀 PM 코치
- 프로젝트 가이드 문서 분석
- 팀 인원 및 기간 입력 시
  - 역할 분담
  - 마일스톤 일정
  - 잠재적 리스크 제안

---

## 5. 사용한 Upstage API

| API | 역할 |
|---|---|
| Solar | AI Agent의 계획 생성 및 피드백 |
| Document Parse | PDF 문서 구조화 |
| Information Extract | 문서 내 핵심 정보(JSON) 추출 |

본 프로젝트는 **Solar + Document Parse + Information Extract를 함께 사용하는 AI Agent 서비스**를 구현하는 데 초점을 두었습니다.


## 8. 교육 콘텐츠로의 확장 가능성   
본 서비스는 교육 환경에서 다음과 같은 방향으로 확장될 수 있습니다.

- 대학 과제 가이드 자동 분석 도구

- 캡스톤 디자인 및 팀 프로젝트 PM 보조 도구

- 코딩 과제 및 온라인 강의 과제 코치

- 입문자를 위한 프로젝트 수행 가이드 AI

AI Agent가 단순히 질문에 답변하는 것이 아니라
문서를 기반으로 계획을 생성하고 피드백을 제공하는 행동을 수행한다는 점에서
기존 챗봇보다 Agent 서비스로서의 확장 가능성이 높다고 판단했습니다.

## 9. 한계 및 향후 개선 방향

- Information Extract 결과의 품질은 문서 형식 및 구성에 따라 달라질 수 있음

- 현재는 단일 문서를 기준으로 한 분석만 지원

- 향후 다중 문서 비교, 일정 관리 도구 연동 등으로 확장 가능
