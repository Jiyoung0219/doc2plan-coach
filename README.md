# 📄 Doc2Plan Coach  
과제·프로젝트 문서를 이해하고, 개인 과제 코칭과 팀 프로젝트 PM 계획을 자동으로 생성하는 AI Agent 프로토타입

---

## 1. 프로젝트 소개

Doc2Plan Coach는 과제 안내 PDF나 프로젝트 가이드 문서를 업로드하면  
문서 내용을 구조적으로 이해하고   
- 개인 과제 수행을 위한 체크리스트 및 계획
- 팀 프로젝트를 위한 역할 분담과 일정 계획  
을 자동으로 생성해주는 AI Agent 서비스입니다.

Upstage의 **Document Parse**, **Information Extract**, **Solar LLM**을 함께 활용하여  
단순 요약이 아닌 **“문서를 읽고 판단하고 계획하는 Agent”**를 구현하는 것을 목표로 했습니다.

---

## 2. 문제 정의

대학 수업이나 프로젝트에서 제공되는 과제 안내 문서는 대부분 PDF 형태이며   

- 요구사항이 길고 흩어져 있음
- 평가 기준이 명확히 구조화되어 있지 않음
- 개인 과제와 팀 프로젝트의 기준이 섞여 있음

이로 인해 과제를 처음 접하는 학생들은  
**“뭘 해야 하는지”, “어디까지 하면 되는지”**를 파악하는 데 많은 시간을 소모합니다.

---

## 3. 해결 방안

이 프로젝트는 다음과 같은 흐름으로 문제를 해결합니다.

1. **Document Parse**  
   - 업로드된 PDF 문서를 AI가 읽을 수 있는 구조화된 텍스트로 변환

2. **Information Extract**  
   - 과제 요구사항, 제출 형식, 평가 기준, 마일스톤 등을 JSON 형태로 추출

3. **Solar LLM**  
   - 추출된 구조 정보를 바탕으로  
     - 개인 과제 체크리스트 및 작업 계획 생성  
     - 팀 프로젝트 역할 분담, 일정, 리스크 분석 생성  
     - 사용자가 작성한 초안에 대해 평가 기준 기반 피드백 제공

---

## 4. 핵심 기능

### 개인 과제 코치
- 과제 요구사항 자동 구조화
- 평가 기준 기반 체크리스트 생성
- 초안/계획/설계에 대한 AI 피드백 제공

### 팀 PM 코치
- 프로젝트 가이드 기반 핵심 목표 추출
- 팀 인원 및 기간 입력 시 역할 분담 제안
- 주차별 일정 및 잠재 리스크 분석
